PacMan

PacMan is modeled after PacMan the arcade version with exact dimensions and functons. I utilized Northwestern's GE211 game engine to customize queries allowing ghosts to move with different characteristics and personalities. I designed one continuous level that support audio, 2D graphics and keyboard interaction. All code is written in C++ and follows the model-view-controller design pattern. PacMan features active and inactive ghosts, big and small points, and a PacMan with three lives.

Trip Planner

Trip Planner implements a trip planning API that provides routing and searching services. To implement this, I utilized graph and dictionary ADTs to establish destinations and their relative connections. I created an efficint O(nlogn) shortest path algorithm to find the shortest path to any destination.

Github

Spotify Music Suggestion
